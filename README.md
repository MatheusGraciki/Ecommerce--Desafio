## Como rodar a aplicação :arrow_forward:

No terminal, clone o projeto: 

```
git clone https://github.com/MatheusGraciki/Ecommerce--Desafio.git
```
Entre na pasta:
```
cd Ecommerce--Desafio
```
Instale as dependencias:
```
npm install
```
Inicie o projeto
```
npm start
```
